AgroPort AI Landing Page
========================

Files included:
- index.html
- style.css
- script.js
- assets/logo.svg
- assets/hero-banner-4k.webp  (primary 3840px-wide hero banner)
- assets/hero-banner-4k.jpg   (fallback 3840px-wide hero banner)
- assets/hero-agro-ai.svg
- assets/cold-storage.svg
- assets/marketplace.svg
- assets/dashboard.svg

What changed:
- The homepage hero section now uses the high-resolution Farm-to-Flight banner as a full-width background.
- Added dark green overlay for readable headline text.
- Added responsive layout for desktop, tablet and mobile.
- Added image preload and fetch priority so the hero banner loads faster.

How to use:
1. Upload all files and the assets folder to your hosting/public_html folder.
2. Keep index.html, style.css and script.js in the same folder.
3. Keep all images inside the assets folder.
4. To change the hero banner later, replace assets/hero-banner-4k.webp and assets/hero-banner-4k.jpg with your final image using the same file names.
5. The contact form currently opens the visitor's email app and sends to momen@agroportai.com. For real database/CRM collection, connect the form to backend, Google Forms, HubSpot, Zoho, Airtable, or WordPress form plugin.
